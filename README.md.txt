## ğŸ¯ About the Project

The Student Result Management System is a beginner-friendly Python application designed to help educational institutions manage student records efficiently. It provides a simple command-line interface for adding student details and automatically calculating their academic performance.

This project is perfect for 1st-semester computer science students as it demonstrates:
- Database operations with SQLite
- Object-oriented programming concepts
- Error handling and input validation
- Clean code structure and organization
- Professional project organization

- **Add Student Details**: Enter student name and marks for 3 subjects

  - Automatically calculates percentage
- **Data Persistence**: Stores all data in SQLite database
- **Input Validation**: Validates all user inputs to prevent errors
- **Professional Structure**: Well-organized code with proper folder structure
- **Error Handling**: Comprehensive error handling for robust operation
